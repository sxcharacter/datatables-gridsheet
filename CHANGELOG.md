# Changelog

All notable changes to the GridSheet project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-03-05

### Added
- New `saveMode: 'basic'`: UI-only mode that skips server/localStorage persistence, perfect for pure spreadsheet behavior.
- Enhanced Formula columns support with local calculations.
- Clean Standalone Mode demo in `examples/basic.html`.
- Detailed documentation for Triple Modes (Direct, Batch, Basic).



### Added
- Initial Release of GridSheet.
- Excel-like grid editing capabilities for DataTables.
- Copy/Paste support (single cell, multi rows, from Excel).
- Built-in context menu.
- Support for inline validation, formats, and themes.
