GridSheet Demo Basic Mode

Standalone UI-only mode with no server persistence required

Basic Mode: All changes made in the grid are strictly local to the browser session. No AJAX calls are made, and no data is saved to localStorage. Perfect for simple data entry forms or data processing tools that handle saving via external logic.

Properties of Basic Mode:

  • No endpoints configuration required.
  • Changes are applied directly to the DataTables internal data.
  • Pasting multiple rows generates temporary IDs automatically.
  • Supports Formula Columns locally (e.g., New Salary = Salary + Increase%).
  • Ideal for scenarios where you want a "Spreadsheet UI" but handle data yourself.
