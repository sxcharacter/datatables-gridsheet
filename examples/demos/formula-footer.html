Formula & Footer Totals Demo

Auto-calculated columns with footer aggregate functions (SUM, AVG, COUNT)

Features: Formula columns calculate values automatically based on other columns. Footer row displays aggregate totals that update in real-time.
☰ Column Legend ▼
Editable - user can edit
Formula - auto-calculated (readonly)

Editable Columns

  • item - Product name (text)
  • qty - Quantity (number)
  • price - Unit price (currency)
  • discount - Discount percentage

Formula Columns

  • Subtotal = Qty × Price
  • Disc. Amount = Subtotal × Discount%
  • Total = Subtotal - Disc. Amount
☰ Footer Aggregates ▼

Aggregate Functions

  • sum - Total of all values
  • avg - Average (2 decimal places)
  • count - Number of non-empty rows

Footer in This Table

  • Qty - SUM
  • Price - AVG
  • Subtotal, Disc. Amount, Total - SUM

Try it: Edit Qty or Price values and watch the formula columns auto-calculate. Add new rows at the bottom and see the footer totals update instantly!
