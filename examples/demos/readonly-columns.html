Readonly & Server-Field Auto Update Demo

Readonly columns with automatic server-field population

Features: Readonly columns cannot be edited by users, but can be auto-populated from server response using data-server-field attribute. Perfect for ID, timestamps, invoice codes, etc.
☰ Readonly Column Types ▼
Editable - user can edit
Readonly - cannot be edited
Server-Field - auto-updated from server

Editable Columns

  • name - Employee Name (text)
  • email - Email (email validation)
  • department - Department (select)
  • salary - Salary (currency format)

Readonly + Server-Field

  • employee_id - Auto-generated ID from server
  • invoice_code - Invoice code (e.g., INV-2026-00001)
  • created_at - Timestamp when row created
  • created_by - Username who created
☰ Developer Guide: How to Use ▼

1. HTML Setup - Add data-server-field attribute:

<!-- Basic readonly (no auto-update) -->
<th data-type="readonly">Static Text</th>

<!-- Readonly with server-field auto-update -->
<th data-type="readonly" data-server-field="id">ID</th>
<th data-type="readonly" data-server-field="invoice_code">Invoice</th>
<th data-type="readonly" data-server-field="created_at" data-format="date">Created</th>
<th data-type="readonly" data-server-field="total" data-format="currency">Total</th>

2. Server Response - Include the field values:

// PHP Example
echo json_encode([
    'status' => 'ok',
    'data' => [
        'id' => $newId,
        'tempId' => $meta['tempId'],
        // These will auto-populate data-server-field columns:
        'invoice_code' => 'INV-2026-' . str_pad($newId, 5, '0', STR_PAD_LEFT),
        'created_at' => date('Y-m-d H:i:s'),
        'created_by' => $_SESSION['username'] ?? 'system'
    ]
]);
Tip: The data-server-field value must match the key in the server response. Use data-format to apply formatting (currency, date, percentage, number).

3. Supported Formats:

  • date - Formats as configured date display
  • currency - Adds prefix, thousand separator
  • percentage - Adds % suffix
  • number - Thousand separator only

Try it: Add a new row by filling the empty row at the bottom. After saving, the ID, Invoice, Created, and Created By columns will be auto-populated!
