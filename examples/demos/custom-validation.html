Custom Validation Demo

Example of custom validation using preSave callback

Demo: Try editing data to violate the validation rules below. Save will be cancelled and an error will appear.

Validation Rules:

Name Minimum 3 characters
Email Must contain @company.com
Age Min 18, max 65
Salary Minimum $ 5.000.000
Score Range 0-100
