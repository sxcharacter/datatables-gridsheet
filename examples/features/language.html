Quick Start

Change GridSheet language in 2 steps

Step 1: Include language file 
<!-- Core GridSheet -->
<script src="gridsheet/js/gridsheet.js"></script>

<!-- Language file (after gridsheet.js) -->
<script src="gridsheet/custom/gridsheet.lang.js"></script>
Step 2: Set language in configuration 
const gridsheet = new GridSheet({
    tableSelector: '#myTable',
    
    // Add language option
    language: GridSheetLang.id,  // Indonesian
    
    // ... other options
});
That's it! All context menus, buttons, and dialogs will now display in Indonesian.

Custom Override

Override individual strings without using language file

Override specific strings inline 
const gridsheet = new GridSheet({
    tableSelector: '#myTable',
    
    // Override only specific strings
    language: {
        saveAll: 'Submit All Data',
        reset: 'Clear Form',
        copyRow: 'Duplicate This Row'
    }
});
Tip: You can combine language file with inline overrides. Inline overrides take priority.
Combine language file with overrides 
const gridsheet = new GridSheet({
    tableSelector: '#myTable',
    
    // Use Indonesian, but override Save button
    language: {
        ...GridSheetLang.id,         // Spread Indonesian
        saveAll: 'Kirim ke Server'    // Custom override
    }
});

Adding a New Language

Step-by-step guide to add your own language

Step 1: Edit gridsheet.lang.js

Open custom/gridsheet.lang.js and add your language following the template: 

const GridSheetLang = {
    // Existing languages...
    id: { ... },
    en: { ... },
    
    // Add your new language
    fr: {
        // Context Menu
        copyRow: 'Copier la ligne',
        pasteRow: 'Coller la ligne',
        clearRow: 'Effacer la ligne',
        insertAbove: 'Insérer au-dessus',
        insertBelow: 'Insérer en-dessous',
        readonlyRow: 'Ligne en lecture seule',
        
        // Buttons
        saveAll: 'Sauvegarder',
        reset: 'Réinitialiser',
        cancel: 'Annuler',
        discard: 'Rejeter',
        restore: 'Restaurer',
        clear: 'Effacer',
        
        // Dropdown
        noResult: 'Aucun résultat',
        
        // Modal
        restoreTitle: 'Restaurer les modifications?',
        restoreMessage: 'Vous avez des modifications:',
        restoreNewRow: 'nouvelle ligne',
        clearTitle: 'Effacer toutes les modifications?',
        clearMessage: 'Cela supprimera toutes les modifications.'
    }
};

Step 2: Use your new language

const gridsheet = new GridSheet({
    tableSelector: '#myTable',
    language: GridSheetLang.fr  // French
});
