Overview

Formula columns automatically calculate values based on other columns

Key features:

  • Auto-calculation - Values update automatically when source columns change
  • Readonly - Formula cells cannot be edited directly
  • Tooltip - Hover over formula cells to see the formula
  • Error handling - Shows error codes like Excel (#DIV/0!, #REF!, etc.)

Basic Usage

Define formula columns using the `columns` option in the GridSheet constructor

JavaScript 
columns: {
    // Source columns
    0: { name: 'qty', type: 'number' },
    1: { name: 'price', type: 'currency', format: { prefix: '$' } },

    // Formula column
    2: { 
        type: 'formula',
        formula: 'qty * price',
        format: { prefix: '$' }
    }
}
Note: Formula columns do not require a name as they represent calculated values, but you can provide one if you want to reference them in other formulas.

Footer Totals

Display aggregate values (SUM, AVG, COUNT) at the bottom of your table

Configuration

JavaScript 
// Enable footer in constructor
new GridSheet({
    tableSelector: '#myTable',
    footer: {
        enabled: true,      // Show footer row
        label: 'Total'       // Label in first column
    }
});

Column Configuration

// Define footer aggregation in columns config
columns: {
    0: { name: 'qty', type: 'number', footer: 'sum' },
    1: { name: 'price', type: 'currency', footer: 'avg' },
    2: { name: 'item', type: 'text', footer: 'count' }
}

Aggregate Functions

Function Description Output Example
sum Total of all values in column $ 50.000.000
avg Average with 2 decimal places $ 6.333.333,33
count Number of non-empty rows 3
Tip: Footer totals automatically update when any cell value changes, including formula columns!
