Quick Start

Apply a theme in 2 simple steps

HTML - Include CSS Files 
<!-- 1. Core GridSheet CSS (required) -->
<link rel="stylesheet" href="gridsheet/css/gridsheet.css">

<!-- 2. Theme CSS (choose one) -->
<link rel="stylesheet" href="gridsheet/custom/theme-ocean.css">
That's it! The theme file includes both light and dark mode styles. Dark mode is automatically activated based on system preference or data-theme="dark" attribute.

Dark Mode Implementation

Three ways to enable dark mode

1. Automatic (System Preference)

Dark mode is automatically enabled when user's OS is set to dark mode. No code needed!

2. Manual Toggle with data-theme

JavaScript - Toggle Dark Mode 
// Toggle dark mode
function toggleDarkMode() {
    const html = document.documentElement;
    const currentTheme = html.getAttribute('data-theme');
    
    if (currentTheme === 'dark') {
        html.setAttribute('data-theme', 'light');
    } else {
        html.setAttribute('data-theme', 'dark');
    }
}

3. Force Dark Mode on Load

HTML - Force Dark Mode 
<html data-theme="dark">
  ...
</html>
Priority: data-theme attribute takes precedence over system preference. If data-theme="light", light mode is used even if OS is dark.

Theme File Structure

Understanding theme CSS organization

Section Description
:root { } CSS variables for light mode colors (primary, selection, borders, etc)
Table Styling Header, rows, cells, hover states, focus states
Selection Styling Row selection, column selection, cell range selection
Context Menu Right-click menu styling
@media (prefers-color-scheme: dark) Automatic dark mode styles based on OS preference
[data-theme="dark"] Manual dark mode styles via HTML attribute

CSS Variables Reference

Customize theme by overriding these variables

CSS - Main Variables 
:root {
    /* Primary Colors */
    --dt-primary: #0ea5e9;           /* Main accent color */
    --dt-primary-hover: #0284c7;     /* Hover state */
    --dt-primary-light: rgba(14, 165, 233, 0.08);

    /* Selection Colors */
    --dt-bg-selected: #e0f2fe;       /* Selected cell bg (light) */
    --dt-bg-selected-dark: #bae6fd; /* Darker selection */

    /* Design Tokens */
    --gs-header-bg: #0ea5e9;         /* Table header background */
    --gs-header-text: #ffffff;       /* Table header text */
    --gs-row-hover: #f0f9ff;         /* Row hover background */
    --gs-row-stripe: #f8fafc;        /* Alternating row color */
    --gs-border-color: #e2e8f0;      /* Table borders */
    --gs-border-radius: 12px;        /* Table corner radius */
}
Quick Customization: To change the primary color, just override --dt-primary and related variables in your own CSS file.

Creating a Custom Theme

Step-by-step guide to create your own theme

Step 1: Copy an existing theme

# Copy ocean theme as starting point
cp custom/theme-ocean.css custom/theme-mycompany.css

Step 2: Replace colors

Use find & replace to change the primary color throughout the file:

Ocean Color Replace With Purpose
#0ea5e9 Your primary color Primary / accent
#0284c7 Darker variant Hover states
#38bdf8 Lighter variant Light accent
#1e3a4d Dark tint of your color Dark mode selection

Step 3: Update header comment

/**
 * GridSheet Theme: My Company
 * Corporate branding colors
 * 
 * Usage:
 * <link rel="stylesheet" href="gridsheet/custom/theme-mycompany.css">
 */
