Overview

Column types are defined in the columns config object or via data-type HTML attribute (legacy)

Define columns in JavaScript instead of HTML attributes: 
columns: {
    0: { name: 'name', type: 'text' },
    1: { name: 'email', type: 'email' },
    2: { name: 'salary', type: 'currency', format: { prefix: '$' } }
}

text Text Input

Standard text input for names, addresses, etc.

{ name: 'name', type: 'text' }

textarea Multi-line Text

Long text input with a popup editor.

{ name: 'notes', type: 'textarea' }

email Email

Email input with automatic format validation.

{ name: 'email', type: 'email' }

number Number

Numeric input (integer/decimal).

{ name: 'age', type: 'number' }

currency Currency

Currency format. See Formats.

{ name: 'salary', type: 'currency', format: {...} }

percentage Percentage

Percentage format with % suffix.

{ name: 'score', type: 'percentage', format: {...} }

date Date

Date picker with configurable format.

{ name: 'birthdate', type: 'date', format: {...} }

time Time

Time picker with HH:mm format.

{ name: 'checkin', type: 'time' }

select Dropdown

Searchable dropdown. Options from endpoint or array.

{ name: 'office', type: 'select', options: [...] }

checkbox Checkbox

Toggle true/false with visual checkbox.

{ name: 'active', type: 'checkbox' }

readonly Read Only

Display only, cannot be edited.

{ name: 'created_at', type: 'readonly', readonly: true }

formula Formula

Auto-calculated column. See Formula.

{ type: 'formula', formula: 'qty * price' }

Column Properties

Properties that can be set in each column config object

Property Required Description
name Yes Field name sent to server on save/update
type No Column type (text, email, currency, etc). Default: text
format No Format options for currency, percentage, date types
options No For select type: endpoint URL or static array
readonly No If true, column cannot be edited
formula No For formula type: calculation expression
footer No Footer aggregation: 'sum', 'avg', 'count', 'min', 'max'
serverField No For readonly columns auto-updated from server response
